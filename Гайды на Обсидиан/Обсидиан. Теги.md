# Тэги
Тэги - это ключевые слова, которые позволяют быстро связывать группы заметок, как я и говорил ранее, аналог хештегов из соцсетей

Для создания тэга нужно поставить решеточку, и без пробела начать что либо вводить #кухня вот таким образом

Можно быстро найти все файлы в которых упомянут этот тэг, нажав по нему

Правила использования тэгов
- Нельзя использовать пробел и спец. символы вроде звездочек и вопросительных знаков
- Тэг обязательно должен начинаться с буквы, а не с цифры

Тэги могут быть вложенными, для этого нужно ввести под тэг после косой черты (/) и после родительского тэга. Например #кухня/рецепты 

Мы можем посмотреть список всех тегов и их структуру в панели тегов справа
