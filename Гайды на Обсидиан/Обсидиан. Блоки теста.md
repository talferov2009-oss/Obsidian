# Комментарии
Если вокруг текста поставить 2 знака процента (%), текст будет скрываться в режиме просмотра
%%вот так%%
# Неформатированный текст
Блок кода - обозначение диапазона, в плагинах им вызываются команды
Для использования нужно поставить по 3 апострофа(там где "Ё") с каждой стороны текста
```
ыоварывлоарывлраыораывлраывларывааыв - вот таким образом
```
Также его можно использовать внутри строки, для этого нужен 1 `апостроф`
# Примечания
Можно создать:
- Нумерованные примечания ( ^1 ) 
- Тестовые примечания ^введите текст 
# Цитаты
Используется знак > с пробелом
> Душа без цели задыхается

# Сноска
цитата с квадратными скобками и !note ( > [!note])
>[! note] sdfsdfsdf

Сноски бывают разных типов (см доп инфу в интернете) 
# Спойлер
Если после Сноски поставить знак -, у нас получится раскрываемая сноска (это пригодится в будущем)
 > [!Спойлер]+ вфывфывыфвыфв

# Таблица
Используются Вертикальные палочки ( | ) как вертикальные палочки в таблице, а разделители как горизонтальные ( ___ )
 Таблицы используются редко, т.к. есть плагины, которые позволяют делать это удобнее
