# Списки
Список - это использование строк для перечисления 
Все списки начинаются с новой строки и поддерживают вложенность
Они бывают нескольких видов:

## Обычный список
Обычный список создается с помощью тире ( - ) плюса ( + ) или звездочки ( * ) и пробела после этих знаков
- Это
- Позволяет
- Быстро
- Создать 
- Перечисление

## Нумерованный список
1. Нумерованный список создается при помощи единицы со скобочкой или точкой после нее 
2. Он автоматически добавляет нумерацию для создаваемых элементов списка
	1. НО он не поддерживает нумерацию вложенных элементов (заголовков)
	2. Я не смогу создать нумерацию вроде (1.2.)
## Список задач
- [ ] Создается следующей комбинацией: ( - [ ] ) 
- [ ] Такой список позволяет создавать переключатели, которые можно использовать для напоминаний или отслеживания прогресса выполнения большой задачи. Например для помечания для себя не просмотренных
- [x] ваыва
