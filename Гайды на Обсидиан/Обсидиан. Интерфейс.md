Обсидиан хранилище - это папка. Можно переключатся между проектами и редактировать их вне Обсидиан
Интерфейс состоит из разделов:
- Панель инструментов (Ribbion) 
	- левая панель инструментов, кастомизируемая
- Рабочая область
	- Открытый файл
	- Вкладки/Табы + возможность менять их режим отображения (слайды)
	- Панель с инструментами и информацией сверху
	- Статусбар снизу справа
- Правая скрываемая панель
Важная особенность Обсидиан - его модульность. Я могу перетаскивать отдельные окна и настраивать их расположение на боковых и центральной панели

Найдя оптимальное расположение окон для конкретной задачи, вы можете сохранить его при помощи 

Организация хранилища в обсидиан строится на основе папок и файлов. В хранилище можно использовать файлы всех типов - картинки, док-ты, аудио, видео и прочее. Но основные инструменты созданы для работы с markdown файлами

Можно отмечать отдельные файлы как избранные при помощи Bookmark, для быстрого доступа к ним

В настройках стоит обратить внимание на:
- О программе -> обновления
- Оформление -> Строка заголовка
- Сочетания клавиш
