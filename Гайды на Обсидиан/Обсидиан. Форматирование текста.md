# Форматирование текста
Форматирование - это добавление к нему эффектов **толщины**, *курсива*, ~~зачеркнутости~~ или ==выделенности==
Для **жирного** теста используются двойной символ звездочки ( ** ) или двойное нижнее подчеркивание ( __ ) 
Для *курсива* используется одна звездочка ( * ) или одно нижнее подчеркивание ( _ ) 
Для одновременного жирного и курсивного текста используется 3 звездочки ( *** ) или 3 нижних подчеркивания ( ___ )
Для ~~Зачеркнутого~~ текста используются 2 тильды ( ~~ )
Для ==подсвечивания== ставим 2 равно ( == ) ^d2b2ad

Форматирование можно комбинировать, главное помнить про уровень вложенности списков
