## Команды линукса
- cd - Команда для перехода в директорию (папку)
- ls (list) -  Команда для вывода содержимого папки
- l - сокращенная версия ls
- clear - команда для очистки терминала
- mv (move) - переместить файл/папку
- rm (remove) - удалить файл/папку
- cp (copy) - копировать файл/папку
- touch - создать файл
- nano - редактировать файл
- mkdir (make directory) - создать директорию
- sudo (superuser do) - выполнить от имени администратора
- chown (change owner) - поменять владельца
- chmod (change modification) - поменять права доступа к файлу.
## Опции команд
Опции - это приписки после команд, позволяющие уточнить их работу. Для каждой команды работает свой список опций. 
- --help - универсальная опция, работающая для всех команд. Позволяет увидеть полный список опций, а так же обьяснение работы команды
