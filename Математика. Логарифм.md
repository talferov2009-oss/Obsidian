## Что такое логарифм
Логарифм $\log_a(b)$ это показатель степени, в которую нужно возвести a, чтобы получить b
**a^$\log_a(b)$** = b
Таким образом, $\log_a(b)$ = x; x - это показатель степени, в который нужно возвести число a
## Свойства логарифма
1.  При равных основаниях логарифма, мы можем опустить их и приравнять подлогарифмиуемые выражения. Работает только если перед равно и после равно один логарифм
	$\log_a(b)$ = $\log_a(с)$ => b = c  
2. Сложение/вычитание. При сложении двух логарифмов с одинаковыми основаниями мы можем умножить подлог выражения, при вычитании разделить их
